Cara Import ke Figma

  1. Buka Figma dan buat file baru bernama SMAVN4 Design System
  2. Buat halaman pertama: Design Tokens — copy semua warna, tipografi, spacing, radius, dan shadow
  3. Buat halaman kedua: Components — buat setiap komponen sebagai Figma Component
  4. Buat halaman ketiga: Screens — susun layout berdasarkan wireframe di dokumen ini
  5. Gunakan Auto Layout di Figma untuk mereplikasi flexbox/grid CSS
  6. Set Variables untuk warna agar dark mode toggle bisa diganti dalam 1 klik
  7. Gunakan Component Properties untuk variant button (size, color, state)
  8. Untuk ikon, install plugin Lucide Icons dari Figma Community

10. Figma Setup Guide

Step-by-Step Figma Recreation

  1. Create Variables: In Figma, go to Local Variables and create color variables matching all tokens above. Create 2 modes: "Light" and "Dark".
  2. Set Typography: Add Poppins (300,500,700,800) and Inter (400,500,600) as text styles. Create styles: H1-H6, Body, Small, Caption.
  3. Build Base Components: Create Figma Components for: Button (5 variants x 3 sizes), Badge (6 colors), Input, Select, Textarea, Card, Stats Card, Progress Bar, Modal, Toast, Timeline Item.
  4. Use Auto Layout: All layouts use Auto Layout with gap/padding matching the 8px spacing system. Use "Fill container" for flex:1 elements.
  5. Component Properties: Add boolean properties for "dark mode", "loading", "disabled" states. Add enum properties for "variant" (primary/secondary/danger/outline/ghost) and "size" (sm/default/lg).
  6. Install Lucide Icons: Search "Lucide" in Figma Community plugins and install. Use 18px stroke icons for nav, 22px for stats cards.
  7. Create Screens: Build each screen from the Screen Map section. Use the App Shell layout component as base frame (1440x900 for desktop).
  8. Dark Mode: Switch all color variables to "Dark" mode to instantly preview dark theme on any screen.

Quick Alternative: HTML to Figma Plugin

  1. Install the "HTML to Figma" plugin from Figma Community
  2. Run the app locally with npm run dev
  3. In your browser, open each page of the running app
  4. Use the HTML to Figma plugin to capture the live rendered page directly into Figma
  5. This gives you pixel-perfect copies of every screen with all real data
  6. Then convert the captured layers into proper Figma Components with Auto Layout
